# 🤖 AI Resume Builder & ATS Analyzer

A full-stack AI-powered Resume Builder with ATS analysis built using React, Node.js, Express, MongoDB, and AI APIs.

## 🚀 Features

- User Authentication
- AI Resume Analysis
- ATS Score Checker
- Multiple Resume Templates
- Live Resume Preview
- PDF Download
- Resume Version History
- Company-Specific Resume Suggestions
- AI Interview Questions
- Dashboard

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
- bcrypt

### AI
- OpenAI API (or another LLM API)
